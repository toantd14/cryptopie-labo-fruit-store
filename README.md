# Fruit Store Application

## Setup

### Enviroiment
```sh
$ git clone git@github.com:toantd14/cryptopie-labo-fruit-store.git
$ cd cryptopie-labo-fruit-store

$ cp ./app/.env.example ./app/.env
```
### Docker run
```sh 
$ docker-compose up -d
```

### Postman
```sh
Import ./postman/CryptopieLabo.postman_collection.json
```
